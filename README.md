# WAVES

南京大学软件学院嵌入式综合实践: `WAVE` car work`S`pace.

## 环境配置

初始化工作空间:

```bash
cd /your/path/to/waves/src
catkin_init_workspace
```

编译源码:

```bash
cd /your/path/to/waves
catkin_make
```

设置环境变量:

```bash
cd /your/path/to/waves
source ./devel/setup.[yourshell]

# 例如对于 bash 用户
source ./devel/setup.bash
```
